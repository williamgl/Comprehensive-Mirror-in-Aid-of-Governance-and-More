声明：
资治通鉴成书已经942年，近千年的传承中由于文字抄写的不便和字体以及避讳等问题，各个版本自然会略有不同。好在，资治通鉴不至于如同道德经一样今古版本出现过多差异，本书中的资治通鉴原文，在出现多个版本的时候，均以元代胡三省音注石刻版为准。此版本刻印于1280年左右，是笔者可以找到的距离司马光完书（1084年）最近的版本。
